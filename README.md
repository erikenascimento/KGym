# K-Gym
A web and mobile application focused on gym-goers and fitness enthusiasts.

## Vision
- Allow users to track personal records per exercise
- Allow users to create and follow workout routines
- Enable gyms and trainers to manage students digitally

## Project Structure
- apps/api: Backend (NestJS)
- apps/web: Web app (React)
- apps/mobile: Mobile app (React Native)
- packages/shared: Shared types and utilities
